# Akshay Bhatnagar | Portfolio

A personal portfolio website built with **Next.js 15** (App Router), **Tailwind CSS**, and **React 19**.

**Live:** [akshaybhatnagar.in](https://akshaybhatnagar.in)

## Features

- Responsive single-page layout with smooth scroll navigation
- **Dark mode** toggle with persistence
- **Contact form** that sends emails directly from the site
- Sections: Home, About, Projects, and Contact
- Active section highlighting based on scroll position

## Quick Start

```bash
npm install
cp .env.example .env   # Add EMAIL_USER and EMAIL_PASS for the contact form
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

| Layer   | Technology  | Purpose                          |
|---------|-------------|----------------------------------|
| Framework | Next.js 15 | Server-side rendering, API routes |
| Styling | Tailwind CSS | Utility-first CSS                |
| Email   | Nodemailer  | Contact form email delivery      |
| UI      | React 19    | Component-based interface        |
